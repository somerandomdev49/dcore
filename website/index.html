Description

In the world of DragonCore (or dcore) you will experience the nostalgic feeling of open world games from the 2000s with lower-polygon graphics and simpler gameplay.

We made this project for ourselves, so once it comes out, you can enjoy (hopefully) it for free. Currently, there are 2 and a half (1 + 0.5 + 0.5 + 0.5 = 2.5) developers working on this game.
